# 工作台书页风格调整设计

**当前状态：** 已完成

## 设计结论

将 `NotebookWorkbench` 的外层视觉语言从研究控制台/深色仪表盘方向调整为“仿书页 / 档案页 / 研究手稿”方向：以纸张色背景、墨色文字、衬线标题和版心式分隔建立气质，只改外壳与顶部区，不触及三栏内部业务结构。

## 范围

- 修改 `client/src/views/NotebookWorkbenchView.vue`
- 修改 `client/src/components/notebook-workbench/NotebookTopBar.vue`
- 必要时微调全局基础字体变量

## 视觉要求

- 避免深色科技面板感、数据可视化感和控制中心感
- 背景接近纸面与档案页，而不是纯白后台或深色控制台
- 标题区更像章节标题、页眉或档案抬头，而不是产品工具条
- 边界与分隔更像版心、栏线、注记，而不是组件卡片描边

## 字体方向

- 标题使用衬线气质字体栈
- 正文使用耐读的中文字体栈
- 整体优先“阅读、整理、研究”感，而不是“操作、监控、操控”感
- 切换到书卷气字体后，同步提高基础字号与行高，避免沿用无衬线模板中的偏小字号
- 长期可见的正文、列表、输入框和按钮文案，默认保持在稳定可读级别，不让主体信息落到过小的辅助字号

## 不做的事

- 不修改 SourcesPanel / ChatPanel / StudioPanel 的内部布局
- 不引入新依赖
- 不重构业务逻辑

## 验证

- 运行 `npm run build --workspace client`
