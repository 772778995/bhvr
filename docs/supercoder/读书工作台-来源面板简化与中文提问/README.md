# 读书工作台-来源面板简化与中文提问设计

**当前状态：** 已完成

## 设计结论

`/book/:id` 左侧来源面板继续收缩为“只呈现当前书与必要操作”，去掉自我解释型眉题和长段文案；同时，自动研究不仅回答要中文，记录在“课题研究历史”里的问题文本本身也必须是中文，不能继续混入英文问题。

## 关键决策

1. 左侧面板顶部删除 `Book Archive`、`当前书籍`、`单书模式...` 等说明性文案，保留上传入口和当前书条目本身。
2. 当前书卡片删除“书籍已接入当前工作台...”这类解释性段落，仅保留状态、文件名和必要操作按钮。
3. 自动研究链路在 `nextQuestion()` 成功返回后，若问题不是中文，则立即落回中文 fallback 问题，不把英文问题写入历史。

## 成功标准

- 左侧面板更紧凑，不再出现多余眉题和说明文字
- 当前书卡片信息密度更高，只保留状态、文件名、替换/删除等操作
- 课题研究历史中的新问题文本保持中文
