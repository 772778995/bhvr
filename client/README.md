# 前端客户端

NotebookLM 自动化研究引擎的前端界面，基于 Vue 3 + Vite + TailwindCSS。

> 注意：前端尚未开发，当前仅为项目脚手架。

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

## 构建

```bash
npm run build
```

## 技术栈

- [Vue 3](https://vuejs.org) — 前端框架
- [Vite](https://vitejs.dev) — 构建工具
- [TailwindCSS](https://tailwindcss.com) — CSS 工具类框架
- [shadcn-vue](https://www.shadcn-vue.com) — UI 组件库（计划使用）
